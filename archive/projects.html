Privacy Guard

November 2018

A browser extension that uses a magnetic field sensor to close all incognito tabs when the user's door is opened. Developed and built in 36 hours at CalHacks 5.0, from November 2-4, 2018. I built the mobile application (using Android Studio) associated with the project, which generated a unique ID that the user would enter in his browser to enable the sensor.

Anti-Harrassment Twitter Bot

November 2016

A bot on Twitter that detects words of possible online harassment and takes step to stop harassment by replying to the offending tweet with a warning. Developed and built in 24 hours at GunnHacks 3.0 from November 11-12, 2016. This was built using Python module Tweepy and Twitter's built-in application management for developers.

This project won a sponsor award from HackHarassment.

Created in collaboration with Jack Irish, Dino Maslic, Pranav Rao, and Adam Saleh.
